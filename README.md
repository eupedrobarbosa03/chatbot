# 🤖 ChatBot Rápido e Funcional

Um chatbot simples, leve e funcional criado com HTML, CSS e JavaScript puro. Ele responde automaticamente com base em palavras-chave associadas a linguagens de programação e frameworks.

## 💬 Funcionalidades

- 🤖 Respostas automáticas com base em palavras-chave (linguagens e frameworks).
- 👋 Saudação personalizada na primeira interação.
- 🧩 Estrutura modular, facilitando manutenção e expansão.
- ⚡ Interface rápida, limpa e responsiva.
- ➕ Fácil de adicionar novas palavras-chave e respostas.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

Esse projeto foi criado com o intuito de praticar conceitos essenciais de JavaScript moderno como:

- ✅ Classes e modularização
- ✅ Manipulação do DOM
- ✅ Delay e resposta automática com setTimeout
- ✅ Uso de import/export com ES Modules
- ✅ Organização de mensagens e lógicas em arquivos separados
